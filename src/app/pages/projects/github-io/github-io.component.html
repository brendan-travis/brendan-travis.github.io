The Project

A small portfolio site built with Angular as a way to learn about Angular and try out some frontend magic.

The first iteration was built to get something built as fast as possible as a good base.

From there, it could be iterated further, including more features and complex components.

The initial goal was to create a landing page for my projects and have a place where I could write a small description and if possible, link to a working prototype.

The Tech

The site is built on Angular 12 and uses Angular Material for some quick and elegant components.

Most of the styling is hand rolled but a few utilities and helpers were added from Bulma.

The Future

The site will continue to host my projects as a place to showcase my work.

Plus, I want to continue developing with Angular and create some custom components that could be reused in other applications.
